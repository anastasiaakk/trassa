# Оригинальные иконки ролей (Page3)

Старые ссылки TagJS (`storage.googleapis.com/.../expires_30_days.png`) больше не открываются.

Чтобы вернуть **те же PNG**, что были раньше:

1. Откройте проект в [CodeTea / TagJS](https://tagjs.com) (id проекта: `KMgTjwx8lt`) или исходный Figma-макет «Страница 3».
2. Экспортируйте 4 пиктограммы (контур на тёмном фоне — фон уберётся при сборке).
3. Сохраните сюда как:
   - `b3pnceya.png` — открытая книга (школьник)
   - `66h5rmum.png` — шапка выпускника (студент)
   - `0tenwd9b.png` — здания (подрядчик)
   - `boty0uwi.png` — колонны / институт (гос. институты)
4. Запустите `npm run dev` или `npm run assets:tagjs`.

Либо положите скрин карточки 4-й роли в `public/page3-refs/role4-institution-reference.png` (как для ролей 1–3).
