# Обновление интерфейса портала «ТрассА» (glass v2)

Текст для рассылки, новости на сайте или баннера в админке.

---

## Кратко (баннер / push)

**Новый вид портала** — спокойная teal-палитра, удобная нижняя панель разделов в кабинетах и улучшенная читаемость. Старый интерфейс можно вернуть в настройках администратора. Подробнее: [PORTAL-DESIGN-USER.md](./PORTAL-DESIGN-USER.md).

---

## Для пользователей (полный текст)

Здравствуйте!

Мы обновили внешний вид портала **«ТрассА»**:

- **Современный glass-стиль** — мягкие панели, акценты бирюзового и голубого.
- **Быстрая навигация** — в кабинетах школьника, студента, подрядчика и ассоциаций внизу экрана появилась панель с основными разделами.
- **Светлая и тёмная тема** — переключатель в шапке кабинета, как раньше, но с более ровным контрастом.
- **Шрифты Inter и Onest** — без изменений, привычная читаемость.

Новый вид включён **по умолчанию**. Если что-то кажется непривычным, администратор организации может вернуть классический интерфейс (бордо/синий) в разделе **Настройки** админ-панели или по инструкции в [PORTAL-DESIGN-USER.md](./PORTAL-DESIGN-USER.md).

Спасибо, что пользуетесь порталом!

Команда «ТрассА»

## Для администраторов

| Действие | Где |
|----------|-----|
| Включить / выключить v2 | Админка → **Настройки** → «Новый дизайн портала (glass v2)» |
| Звук dock (только `.exe`) | Там же → «Звук при нажатии в dock» |
| Превью компонентов | `#/design-preview` |
| Чеклист перед релизом | [PORTAL-DESIGN-STABILIZATION.md](./PORTAL-DESIGN-STABILIZATION.md) |
| Откат CSS (разработка) | `node scripts/restore-portal-design.cjs snapshot-…` |

## Технические заметки (внутренние)

- Ключ: `localStorage['trassa-portal-design']` → `v2` | `legacy` | отсутствует (= v2).
- Событие смены: `trassa-portal-design-changed`.
- Тема кабинета: `trassa-cabinet-theme` + `data-cabinet-theme` на `<html>`.
