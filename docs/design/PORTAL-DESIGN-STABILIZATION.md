## Доступность

- Tab по dock, модалкам, полям — видимый `:focus-visible`.
- `prefers-reduced-motion: reduce` — без лишних анимаций.
- `prefers-contrast: more` — усиленные границы (a11y.css).

## Откат

```js
localStorage.setItem('trassa-portal-design', 'legacy');
location.reload();
```

Снимок CSS: `node scripts/restore-portal-design.cjs snapshot-2026-05-22`

## Legacy-регрессия

Повторить ключевые строки таблицы с `legacy` — визуал бордо/синий, без dock.
