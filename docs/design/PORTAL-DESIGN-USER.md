# Новый интерфейс портала (glass v2)

Краткая памятка для пользователей и администраторов.

## Что изменилось

- Спокойная **teal/cyan** палитра вместо бордо/синего в кабинетах.
- **Стеклянные** панели, скругления, нижняя **панель быстрых разделов** (dock) в кабинетах.
- **Светлая и тёмная** тема — переключатель в шапке кабинета (иконка солнца/луны).
- Шрифты **Inter** и **Onest** — без изменений.

## По умолчанию

У новых визитов включён **v2**. Если интерфейс кажется непривычным, можно вернуть классический вид (см. ниже).

## Вернуть старый вид (legacy)

1. Откройте консоль браузера: **F12** → вкладка **Console**.
2. Вставьте и нажмите Enter:

```js
localStorage.setItem('trassa-portal-design', 'legacy');
location.reload();
```

Либо в **кабинете администратора** → **Настройки** → выключите «Новый дизайн портала (glass v2)».

## Снова включить v2

```js
localStorage.setItem('trassa-portal-design', 'v2');
location.reload();
```

Или переключатель в настройках админки.

## Нижняя панель (dock)

В кабинетах **школы**, **СПО**, **подрядчика**, **РАДОР/АДО** внизу экрана — быстрые кнопки разделов. На узком экране панель прокручивается вбок.

## Desktop-приложение «Трасса»

- Звук при нажатии на dock: **Админка → Настройки → «Звук при нажатии в dock»** (только в установленном `.exe`, не в обычном браузере).
- Обновления и установка — см. [DESKTOP.md](../desktop/DESKTOP.md).
